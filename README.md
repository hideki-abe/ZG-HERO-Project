# ZG-HERO-Project
Projeto ZG-HERO do Programa de Aceleração da ZG Solutions.

<h3>ETAPA DE WEB SERVICES E MICROSERVICE</h3>
  Nessa etapa, um serviço de microservice de cadastro teve que ser feito sem a utilização de frameworks. Foram utilizados Sevlets e JSP para a integração entre o backend e o frontend. 
  Alguns <a href="https://github.com/hideki-abe/ZG-HERO-Project/tree/microservice/Jars">jars</a> tiveram que ser colocados manualmente por ocorrência de alguns bugs que estavam acontecendo. 
  
  O Tomcat 10 foi utilizado para o servidor, e basta configurar o servidor nas ferramentas do Intellij e inicializar para que hava a integração do frontend e backend.
